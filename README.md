# Ecommerce Selenium Framework

Automation framework built using:

- Python
- Selenium
- Pytest
- Page Object Model
- Data Driven Testing
- HTML Reports

## Features
- Cross-browser execution
- JSON data support
- Explicit waits
- Pytest fixtures
- Screenshot capture on failures

## Run Tests

```bash
python -m pytest -v -s --html=reports/report.html --browser_name edge
